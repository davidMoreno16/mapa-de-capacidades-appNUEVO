# Mapa de Capacidades – Mega Mayorista

## Información del Estudiante
**Nombre:** David Moreno  
**Materia:** Desarrollo de Aplicación Nativa en el Cloud  
**Unidad:** 2 – Front-End con React + Next.js  

---

## Información de la Empresa
**Empresa:** Mega Mayorista  
**Sector:** Comercio – Venta de artículos electrónicos  
**Descripción:** Mega Mayorista es una empresa dedicada a la comercialización de productos electrónicos y de tecnología, ofreciendo variedad de productos para clientes mayoristas.  
**Objetivo del Proyecto:** Representar de manera visual el mapa de capacidades del negocio mediante una aplicación web moderna en React + Next.js.

---

## Mapa de Capacidades del Negocio
El mapa de capacidades identificado durante el Prácticum 3 incluye:

- **Ventas**
- **Inventarios**
- **Logística**
- **Clientes**
- **Proveedores**

Estas capacidades representan los pilares operativos principales de Mega Mayorista.

---

##  Aplicación Objetivo
Aplicación web desarrollada con **React + Next.js**, cuyo propósito es mostrar de manera organizada y visual el mapa de capacidades del negocio.

### Funcionalidades Principales
- Visualización clara del mapa de capacidades
- Componentes reutilizables
- Diseño responsivo
- Implementación con JavaScript moderno
